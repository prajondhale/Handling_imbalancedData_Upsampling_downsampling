## Handling Imbalanced Data

 I practiced handling imbalanced datasets using both **Upsampling** and **Downsampling** techniques by understanding the tutorial.

* **Upsampling:** Increased the number of samples in the minority class to reduce class imbalance and improve model learning.
* **Downsampling:** Reduced the number of samples in the majority class to create a more balanced class distribution.

The objective of applying these techniques was to understand how class imbalance affects model performance and to compare different balancing approaches for improving prediction quality and reducing bias toward the majority class.

This practice helped in learning:

* Data preprocessing techniques
* Class balancing methods
* Impact of imbalance on model performance
* Dataset preparation before model training
